# Fungal Evolution

A multi-stage incremental game built with React + Vite.

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Game Modes

-   **Spore Cluster**: Click the mushroom to gather spores. Buy upgrades to evolve.
-   **Mycelium Garden** (Unlocks at 100 Spores): Plant seeds and grow fungal warriors.
-   **Warrior Missions** (Unlocks at 1 Warrior): Send your warriors to gather resources from the land.
